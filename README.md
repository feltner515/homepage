# Research Homepage

A professional research website built for GitHub Pages.

## Features

- **Homepage**: Welcome section with introduction
- **About**: Personal and professional background
- **Research**: Overview of research areas and projects
- **Publications**: List of academic publications
- **Contact**: Contact information and social links

## Viewing the Website

This website is designed to be hosted on GitHub Pages. Once deployed, it will be accessible at:
`https://[username].github.io/homepage/`

## Local Development

To view the website locally, simply open `index.html` in a web browser.

## Customization

Edit the following files to customize the website:
- `index.html`: Main content and structure
- `style.css`: Styling and design
- `_config.yml`: GitHub Pages configuration

## GitHub Pages Setup

1. Go to repository Settings
2. Navigate to Pages section
3. Select the branch to deploy (usually `main` or `master`)
4. Save and wait for deployment

The website will be live in a few minutes!
